# 專案練習 - BeautyOne 電商平台

BeautyOne 是一個以 Vue 3 (Vue CLI) 打造的電商網站專案，模擬完整購物流程，包含商品瀏覽、收藏、加入購物車、結帳流程等功能。
後端資料由課程提供的 線上 API (Swagger 介面) 支援，練習前後端串接、狀態管理與頁面動態呈現。

此專案主要用於練習：

- 前端介面切版與使用者體驗設計
- 商品清單、收藏、購物車等功能邏輯管理
- 前端表單驗證
- 與後端 API 的 CRUD 串接
- 完整的電商結帳流程

使用者可以在平台中體驗完整的購物流程：

- 查看所有商品與分類
- 查看商品詳情
- 將商品加到「我的最愛」
- 新增商品至購物車
- 使用優惠券
- 完成「填寫資料 → 確認付款 → 完成訂購」完整流程

---

## 使用技術

### Frontend

- Vue 3
- Vue CLI
- Vue Router
- Axios（API 串接）
- Option API
- VeeValidate（表單驗證）
- SCSS / RWD 設計
- Bootstrap 5

### Backend（課程提供 API）

#### 六角學院電商 API（Swagger 文件）

- 商品 CRUD
- 購物車管理
- 訂單建立
- 優惠券領取
- 使用者登入驗證
